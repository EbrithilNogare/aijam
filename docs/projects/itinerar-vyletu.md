# Itinerář výletu

## Prompt pro Google Stitch

Navrhni interaktivní itinerář třídenního výletu po jižní Moravě. Stránka by měla zobrazovat časovou osu s jednotlivými zastávkami — vinné sklepy, zámek Lednice, Pálava a historické centrum Znojma. Ke každé zastávce přidej kartu s fotkou, krátkým popisem a odhadovaným časem. Celkový vzhled by měl evokovat cestovatelský deník s teplými barvami a ručně kreslenými ikonami.
