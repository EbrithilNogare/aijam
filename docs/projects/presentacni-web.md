# Presentační web

## Prompt pro Google Stitch

Vytvoř moderní jednostránkový presentační web pro fiktivní kreativní studio "Atelier Luma". Web by měl obsahovat hero sekci s výrazným nadpisem a animovaným pozadím, sekci s nabídkou služeb (branding, webdesign, fotografie), galerii realizací a kontaktní formulář. Použij elegantní typografii, jemné přechody barev a minimalistický design s důrazem na bílý prostor.
